## Changes

This bundled version was adapted for the AITuber OnAir Inochi2D example.

Changes include:

- Rig adjustments for browser-based Inochi2D rendering.
- Additional helper rig controls for avatar motion.
- Idle motion data adapted for the sample app.
- Motion metadata configured through `public/inochi2d/manifest.json`.

The original model attribution and license are preserved above.
